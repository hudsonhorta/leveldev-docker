# leveldev-docker
Aplicação em Ruby rodando dentro de um Docker para o curso de LevelDev da Locaweb

Projeto de criação de uma aplicação no formato do Twitter, para consumo de API utilizando tecnologia Docker e Ruby e .Vue

A pasta \client\ é a parte em frontEnd criada em .Vue para consumo da API

A pasta \web\ é a parte de backEnd criada em Ruby

ps: por questão de tempo, não foi possível concluir a parte visual do projeto, ficando para uma futura atualização.
